# 🚀 AICraft

AICraft is a web-based AI-powered application designed to provide intelligent features through a clean and interactive interface.  
The project combines a Python backend with a dynamic frontend to deliver AI-driven functionality in a simple and scalable way.

---

## ✨ Features

- 🤖 AI-based processing and logic
- 🌐 Modern web interface
- ⚡ Fast Python backend
- 🗂 Organized architecture with templates and static assets
- 🔌 Deployment-ready configuration
- 📦 Easy setup and installation

---

## 🛠 Tech Stack

### Backend
- Python
- Flask

### Frontend
- HTML
- CSS
- JavaScript

### Deployment
- Gunicorn
- Procfile support

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/game8527690-rgb/aicraft.git
```

Move into the project directory:

```bash
cd aicraft
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Run locally:

```bash
python main.py
```

Run with Gunicorn:

```bash
gunicorn app:app
```
